## Getting Started

### Prerequisites

- pnpm
  ```sh
  brew install pnpm # Or npm install -g pnpm
  ```

### Installation

1. Clone the repo

```sh
git clone https://github.com/JawilX/jtools.git
```

2. Install NPM packages

```sh
pnpm install
```

3. Run

```sh
pnpm dev
```
